# Character networks model plot as space, not time

Source: Moretti 2011, "Network Theory, Plot Analysis"

When you turn a dramatic text into a character network (characters = vertices, interactions = edges), you lose temporality but gain spatiality. The past becomes as visible as the present — once a character appears in the network, they never disappear. "What is done is never undone."

This is a fundamental trade-off: networks sacrifice the sequential experience of plot (this happens, then that) for a simultaneous view of all relationships. The advantage is that you can now see **regions**, subsystems, and structural patterns that are invisible when reading linearly.

Moretti frames this explicitly: "Four hours of action, that become this. Time turned into space: a character-system arising out of many character-spaces."

This connects to Alex Woloch's concept of "character-space" from *The One vs. the Many* — each character occupies a portion of the narrative's attention, and the network makes these portions visible simultaneously.

## Implications for tvplotlines

- Our pipeline extracts storylines from synopses — a temporal representation. Network analysis could complement this by revealing the spatial structure of character relationships.
- The time→space trade-off means network analysis and sequential storyline extraction answer different questions about the same text.
