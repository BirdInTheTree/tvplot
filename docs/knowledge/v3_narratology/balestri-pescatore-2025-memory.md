# Narrative Memory in Machines: Multi-Agent Arc Extraction in Serialized TV

## Summary

Расширение ICAART-статьи. Вводит архитектуру памяти по аналогии с человеческой: LLM как семантическая память (общие нарративные паттерны), векторная БД как эпизодическая память (конкретные арки), мульти-агентный workflow как рабочая память.

## Key concepts

- **Episodic memory** — хранение конкретных арок и их прогрессий в векторной БД
- **Semantic memory** — LLM как источник общих нарративных паттернов
- **Working memory** — мульти-агентный процесс как аналог рабочей памяти
- **Computational memory architecture** — формализация нарративного понимания через модели памяти

## Relevance to tvplotlines

Самый близкий аналог. Тот же домен (сериалы), те же авторы (Balestri & Pescatore), но другая теоретическая рамка — когнитивная (память), а не сценарная (драматические функции). Интересно как альтернативный фреймворк.
