# Events are transitions between states, not actions by characters

Source: Bal 1988, "Narratology", Ch. 3.2 Events (p. 155)

Bal defines an event as "the transition from one state to another state." The word "transition" stresses that an event is a *process*, an alteration — not a static fact and not necessarily an action.

Compare "John is ill" (a condition — not an event) with "John falls ill" (a transition — an event). The difference is in the verb: the second implies a change of state.

This is subtly but importantly different from the tvplotlines definition: "One action by one character that changes the situation." Bal's definition is broader: an event can be *experienced* (not just *performed*), and the focus is on the state-change, not on the action itself.

## Consequence

"John falls ill" is an event even though no one caused it — it's a transition experienced by an actor. This matters for TV analysis because many significant plot events are things that *happen to* characters (a diagnosis, a discovery, an accident), not things characters *do*.
