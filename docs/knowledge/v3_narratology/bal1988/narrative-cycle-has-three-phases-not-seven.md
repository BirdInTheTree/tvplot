# Narrative cycle has three phases, not seven

Source: Bal 1988, "Narratology", Ch. 3.2 Events (pp. 160-164)

Instead of Freytag's linear arc (setup → inciting_incident → escalation → turning_point → crisis → climax → resolution), Bal presents Bremond's model: the fabula is a grouping of **elementary series**, each with three phases:

1. **Possibility** (virtuality) — a potential action or change
2. **Realization** — the action happens (or doesn't)
3. **Result** (conclusion) — success or failure

None of these phases is obligatory. A possibility can remain unrealized. A realization can lack a clear result. This is fundamentally different from Freytag: there's no prescribed shape, no required arc.

## Direction: improvement or deterioration

Each cycle has a direction:

```
Improvement to be achieved → process of improvement → improvement achieved
                           → no process            → improvement not achieved

Foreseen deterioration      → process of deterioration → resultant deterioration
                           → no process               → deterioration avoided
```

This replaces "escalation" and "turning_point" with something more informative: the **direction** of change. "Escalation" tells you intensity is increasing but not *which way*. Bremond tells you: is this an improvement process or a deterioration process for the subject?

## Embedding

Series nest inside each other. The result of one series becomes the possibility of another. One realization can open a new possibility:

```
John is tired (possibility: rest)
  He rests → feels fine (improvement achieved)
    Feels fine = can work (new possibility)
      Works → gets tired (deterioration)
```

Or more dramatically:

```
John is tired (possibility: sleep)
  Falls asleep (realization)
    = possibility: forget exam (embedded deterioration)
      Forgets → fails exam (deterioration achieved)
John feels fine again (primary improvement achieved)
```

The primary series (improvement) and embedded series (deterioration) can have different importance. The embedded one may matter more for the fabula.

## What this means for tvplotlines

Instead of asking "is this event setup, escalation, or climax?", we could ask:
1. **Direction**: improvement or deterioration (for which actor)?
2. **Phase**: possibility, realization, or result?
3. **Is this primary or embedded** in another series?

This gives richer information. "Walt strangles Krazy-8" is: realization of deterioration (elimination/sacrifice), embedded in a primary improvement series (Walt's survival).
