# Initial situation is always a state of deficiency

Source: Bal 1988, "Narratology", Ch. 3.2 Events (p. 164)

"The initial situation in a fabula will always be a state of deficiency in which one or more actors want to introduce changes."

This is a strong claim: every fabula begins with something lacking, something wrong, something that needs to change. The development reveals whether the change process leads to improvement or deterioration relative to this initial state.

## Relation to Todorov

This aligns with Todorov's equilibrium model (equilibrium → disruption → new equilibrium), but with an important difference: Bal/Bremond don't assume the initial state is an *equilibrium*. It's already a deficiency. The fabula is the attempt to fix (or worsen) it.

## What this means for tvplotlines

Our "story_engine" and "inciting_incident" implicitly assume a stable starting point that gets disrupted. But many TV shows start *in medias res* — the deficiency is already present in S01E01 (Walter White already has cancer before the pilot's midpoint). By Bal's logic, the "inciting incident" isn't what creates the deficiency — it's the moment the subject gains **determination** (competence to act). This is a subtle but useful reframe.
