# Эксперимент: русские vs английские промты для русских синопсисов

## Гипотеза

По данным AlKhamissi et al. (2024), язык промта активирует культурно-специфичные знания модели. Русский промт при анализе русских синопсисов должен давать более точный результат, чем английский — потому что активирует «подсеть» русской литературной и сценарной традиции.

## Дизайн

- Взять русский сериал с русскими синопсисами
- Прогнать через tvplotlines дважды: `--lang ru` и `--lang en`
- Сравнить результаты: качество сюжетных линий, точность Story DNA, адекватность event functions
- Оценка: ручная (экспертная) + количественная (количество storylines, confidence levels, патчей из Pass 2, вердиктов из Pass 3)

Ждёт выполнения. Зависит от стабильности pipeline.
