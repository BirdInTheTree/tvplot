# Fine-tuning локальной модели для Pass 2

## Зачем

Pass 2 (назначение событий на сюжетные линии) — самый частый LLM-вызов в пайплайне: один вызов на каждый эпизод. Fine-tuning маленькой модели (Qwen 7B) позволит запускать Pass 2 локально, бесплатно, без API. Остальные проходы остаются на Claude (они дешёвые — 1–3 вызова на сезон).

Без fine-tuning локальная модель тупит (проверяли) — не понимает narratology-терминологию, ломает JSON. С fine-tuning она учится воспроизводить паттерн из примеров Claude (knowledge distillation).

## Предпосылки

- Промпты должны быть **стабилизированы** — каждое изменение промпта обесценивает training data
- Все старые результаты (House, ER и др.) получены на устаревших промптах (другая схема: `driver` вместо `hero`, `franchise_type` вместо `format`, нет `nature`/`confidence`/`is_anthology`, нет Pass 4)
- Breaking Bad S01 ближе к текущему формату, но тоже есть расхождения (`plotline` vs `plotline_id`)
