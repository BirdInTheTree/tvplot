# Enculturation-аргумент tvplotlines: центральная идея

LLM уже несёт в себе нарративные нормы из обучающих данных — паттерны сюжетной структуры, представления о «правильном» развитии истории, типичные арки персонажей. Промты tvplotlines не учат модель этим вещам с нуля. Они **активируют и направляют** уже усвоенные паттерны, задавая систему координат (Story DNA, franchise types, event functions), через которую модель структурирует своё понимание.

## Откуда это следует

- **Frazier et al. (2019):** модель, обученная на детском комиксе, переносит нормы на Star Wars без дообучения. Нарративные нормы усваиваются из историй и переносятся.
- **Jung et al. (2026):** LLM «от природы» предпочитают стиль сюжету (на 67% чаще). У модели уже есть нарративные предпочтения до всяких промтов — промты их перенаправляют.
- **AlKhamissi et al. (2024):** язык промта активирует культурно-специфичные «подсети» знаний. Промт — переключатель, не учебник.
- **Winston (2011, 2017):** культура системы = набор правил вывода и концептуальных паттернов. Enculturation = передача этого набора.
- **Piper & Bagga (2024):** теория → словарь понятий → промт → результат. Промт операционализирует теорию, давая модели vocabulary для структурирования того, что она уже знает.
- **Liu et al. (2026):** LLM как «нарративный артефакт» — хранилище нарративных паттернов человечества.

## Следствия для промтов tvplotlines

1. Промты — не инструкции для неучёного работника. Это **система координат для эксперта**, который уже знает предмет, но которому нужно сказать, в какой рамке работать.
2. Разные промты = разные «культуры анализа» = разные результаты из того же текста (Winston: Dr. Jekyll / Mr. Hyde).
3. Язык промта имеет значение: русский и английский промты могут активировать разные нарративные паттерны (AlKhamissi).
4. Модель имеет собственные предпочтения (style bias по Jung) — промты должны это компенсировать.
