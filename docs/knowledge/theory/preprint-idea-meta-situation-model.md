# Мета-situation model: расфокусированное понимание потока

## Наблюдение

Maryanne Wolf и другие говорят: скроллинг убивает глубокое чтение — нет времени на инференсы, эмпатию, критический анализ одного текста.

Но есть другой эффект: информационный поток настолько интенсивен, что заметить тенденции и смыслы можно именно **расфокусировавшись** и глядя на поток целиком, а не на отдельное сообщение.

## Два режима понимания

**Фокусированное чтение** — один текст, глубоко. Строишь situation model конкретного нарратива: персонажи, мотивации, причинность, подтекст.

**Расфокусированное скольжение** — поток текстов, каждый поверхностно, но видишь мета-паттерн. Строишь situation model не текста, а **информационной среды**: «все стали злее», «эта тема вытеснила ту», «этот нарратив повторяется из разных источников».

В терминах Zwaan: первый режим строит situation model **внутри** текста. Второй строит **мета-situation model** — модель потока текстов.

## Связь с plotter

tvplotlines работает во втором режиме: не читает один эпизод глубоко, а скользит по потоку эпизодов и замечает паттерны — эта линия растёт, эта угасает, тут тематическая рифма, тут конвергенция. Pass 3 — буквально «расфокусированный взгляд на целый сезон», видящий структуру, невидимую из одного эпизода.

## Важная оговорка

Скроллинг **предоставляет возможность** для мета-situation model, но пока никто не освоил этот режим осознанно. Люди скроллят — но не строят мета-модели целенаправленно. Возможность есть, навыка нет.

## Статус

Сырая идея. Пока неясно, что с этим делать — может быть discussion-секция препринта, может отдельная заметка, может ничего.
