# Standalone HTML Viewer — Design Decisions

## Решённое

1. **Формат** — один скачиваемый HTML-файл. Открыл в браузере — работает. Как Excel-таблица.
2. **Функционал** — всё что есть в tvplotlines-app: грид (сюжетные линии x эпизоды), аналитика, редактирование (merge/split/edit событий и линий).
3. **Целевая аудитория** — сценаристы 55+, минимальные технические навыки. Ноль терминала.
4. **Безопасность данных** — данные не покидают компьютер пользователя. Никакого нашего сервера. Пользователь доверяет своему LLM (Claude/ChatGPT), но не стороннему сервису.
5. **Онбординг** — при первом открытии HTML проигрывается демо "в реальном времени" — анимированная симуляция пайплайна на примере Breaking Bad. Проход за проходом, результаты появляются на экране. Есть кнопка "посмотреть онбординг заново".
6. **Демо-данные** — Breaking Bad, встроены в HTML.
7. **Связь с библиотекой** — `tvplotlines run --html` генерирует такой же HTML с результатами пользователя.

8. **LLM доступ** — два варианта на выбор пользователя:
   - **Свой API ключ** — бесплатно, вставил Claude/OpenAI ключ и работает
   - **Через нас** — платный аккаунт (5-10 евро), логин/пароль, ничего настраивать не надо. Нужен прокси-бэкенд с нашим API ключом.
9. **Обработка синопсисов** — прямо в HTML. Загружаешь txt-файлы, LLM обрабатывает, результат появляется в интерфейсе.
10. **Онбординг-мастер** — пошаговый wizard при первом запуске: выбери AI → вставь ключ (с картинками-инструкциями) / или залогинься через нас.

11. **Видео-онбординг "как получить API ключ"** — отдельное видео/анимация с пошаговой инструкцией.

18. **Распространение** — HTML viewer живёт внутри библиотеки tvplotlines (один пакет):
    - Для сценаристов: скачать viewer.html с сайта/GitHub (кнопка Download)
    - Для разработчиков: `pip install tvplotlines` → `tvplotlines run --html`
    - tvplotlines-app (FastAPI) остаётся как dev-инструмент для редактирования

## Будущие фичи (не в первой версии)

- **Платный аккаунт** — прокси-бэкенд, оплата, логин
- **Screen assistant как онбординг-помощник** — AI видит экран пользователя и подсказывает что делать дальше (как Clicky, но для tvplotlines). Отдельный проект.

12. **Экспорт** — три формата:
    - XLS — таблица с гридом
    - TXT — простой текст
    - Final Draft (.fdx) — XML-формат, родной для сценаристов. Killer feature.

13. **V1 = просмотр + экспорт**. Редактирование (merge/split/правка) — в следующей версии.
14. **Welcome экран** — диджитальный аватар автора, машет рукой, кнопка "Play me".
15. **Онбординг → LLM подключение** — в конце демо сразу появляется окно "подключить свой LLM" (не отдельный шаг).
16. **Экспорт** — XLS, TXT, Final Draft (.fdx), JSON (наш формат tvplotlines).
17. **Накопление результатов** — все обработанные сериалы сохраняются в localStorage. Можно зайти потом, переключаться между сериалами.
